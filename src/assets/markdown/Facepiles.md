## Facepiles

This prototype tests the inclusion of face piles (user profile avatars in a circle) that appear in real-time when users engage with a post. There are three different parameters with two options each:

- Parameter 1: Facepiles On/Off
    - A: Facepiles On
    - B: Facepiles Off
- Parameter 2: Facepile Direction
    - Y: Left to Right
    - Z: Right to Left
- Parameter 3: Facepile Animation
    - O: Animation On
    - P: Animation Off
